# 🤖 ESP32 Surveillance Robot with Robotic Arm

## 📌 Overview
This project is an **app-controlled surveillance robot** built using **ESP32**, integrated with a **camera module** and a **robotic arm**.  
It is designed for **real-time monitoring and remote object handling** in hazardous or inaccessible environments.

---

## 🚀 Features
- 📱 Wireless control via mobile app (WiFi)
- 🎥 Live video streaming using ESP32-CAM
- 🤖 Robotic arm for pick-and-place tasks
- 🔄 Real-time movement (forward, backward, left, right)
- 💡 Low-cost and scalable design

---

## 🧠 Problem Statement
Human involvement in hazardous environments such as **fire accidents, disaster zones, and bomb disposal** is risky.  
This project provides a **safe and remote-controlled robotic solution** for surveillance and interaction.

---

## 💡 Proposed Solution
A smart robot that:
- Streams live video
- Moves based on user commands
- Interacts with objects using a robotic arm
